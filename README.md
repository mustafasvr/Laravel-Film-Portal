## [Proje/Başlangıç/Altyapı]
PHP tabanlı **api** kullanılarak film öneri puanlama tavsiye portalı

| Açıklama | Yapı |
| ------ | ------ |
| Kullanılacak framework | Laravel |
| Kullanılacak database | Mysql |
| Kullanılacak film api | TMDB |
